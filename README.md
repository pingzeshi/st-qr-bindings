# Quick Reply Bindings

A SillyTavern third-party extension that binds the currently edited Quick Reply set to world books and presets.

## Requirements

- SillyTavern Quick Replies extension
- ST API Wrapper extension at `third-party/st-api-wrapper`

## Usage

1. Open SillyTavern Extensions.
2. Open Quick Replies.
3. Scroll to Edit Quick Replies.
4. Select the Quick Reply set from the dropdown.
5. Use the binding toolbar to bind world books or presets.

When an active world book or preset matches a binding, the bound Quick Reply set is added and shown. When it no longer matches, the bound Quick Reply set is removed from active global/chat Quick Reply sets.
